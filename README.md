## Projects

### Personal Diary CLI (`project1.py`)
A command line diary application where users can write, read and manage diary entries stored in a file.

### Phonebook with Validation (`project2.py`)
A phonebook application that stores contacts with name and phone number validation to prevent invalid entries.

### Student Grade Logger (`practice2.py`)
Logs student names and grades using classes. Data is saved to a file and exceptions are handled for invalid input.

### Bank Transaction Logger (`practice3.py`)
Simulates basic bank transactions — deposits and withdrawals — with exception handling and file logging.
